# 🎥 Reezo

A modern real-time video conferencing platform built with the MERN stack and LiveKit, enabling secure, high-quality video meetings with real-time collaboration.

---

## ✨ Features

- 🔐 Secure User Authentication
- 🎥 HD Video & Audio Meetings
- 👥 Create & Join Meeting Rooms
- 💬 Real-Time Chat
- 🎤 Microphone Controls
- 📷 Camera Controls
- 🖥️ Screen Sharing
- 👥 Participant Management
- 📱 Fully Responsive Design
- 🎨 Modern Glassmorphism UI
- ⚡ LiveKit Integration
- 🌙 Premium Dark Theme

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Virajkamble632/Reezo.git
cd Reezo
```

### Install Frontend

```bash
cd frontend
npm install
npm run dev
```

### Install Backend

```bash
cd backend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **backend** folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

LIVEKIT_API_KEY=your_livekit_api_key

LIVEKIT_API_SECRET=your_livekit_api_secret

CLIENT_URL=http://localhost:5173
```
